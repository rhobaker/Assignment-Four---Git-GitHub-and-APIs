requirements.txt

What is requirement.txt

It is a list of all of a project's dependencies i.e. the versions of all the modules etc. needed to run the repository.

This means it is a list of pip install arguments in a text file.

This enables both yourself and other people to know what you need to install in a virtual environment to run the files in the repository

How to create requirements.txt

Run the command:

pip3 freeze

This will produce a list of everything installed in the virtual environment the repository is running in.
