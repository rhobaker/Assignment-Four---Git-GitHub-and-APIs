requirements.txt

What is requirement.txt?

The requirement.txt file is a list of all of a project's dependencies i.e. the versions of all the modules etc. needed to run the files in the repository.

This means it is a list of pip install arguments in a text file.

It enables both yourself and other people to know what needs to be installed in a virtual environment to run the files.


How to create requirements.txt

In the terminal, while the virtual environment is open, run the command:

pip3 freeze


This will produce a list of everything installed in the virtual environment the repository is running in.
